GUD BOI Documentation

Author: Năstasă Ioana; Group: 3A3

Introduction

This documentation serves as a comprehensive guide to understanding and using the features of the GUD BOI app, available on both mobile devices and smart collars. GUD BOI is a dog care app, which aids in monitoring a dog's health, accessing training resources (AR training or pre-recorded videos), and connecting with other dog owners.

Set up

For mobile and smart collar:

  1. Input Dog Information: User enters essential details about their dog: name, breed, age, weight, and other relevant health information (health issues, special nutrition etc.).
  2. Owner Contact Information: Owners provide their contact details (name, phone number, address) to facilitate communication in emergencies.

For mobile only:

  1. Customization of the app experience is possible by configuring preferences: notification settings, preferred training methods, and community interaction settings

User interface

Mobile design patterns

  • Navigation - drawer menu: most of the features are encompassed in the drawer menu: dog profile page, statistics, community section, pet shop section.
  • Layout - cards: the most commonly used features of the app are displayed as cards, for simplicity and readability: log day (add caloric intake, add exercise levels, add medicine for the current day, add other info), training section, recommendations
  • Actions and commands - tap, swipe, pinch, spread, pan, flick
  • Form patterns - label, input, placeholder, warnings

Smart collar design patterns

  • Navigation - swipe up/down: users navigate through the menu options (contact information, health, meals, training, settings) by swiping through them vertically on the smart collar screen.
  • Layout - single screen: due to space constraints, all features are presented on a single screen layout, ensuring ease of access
  • Actions and commands - tap
